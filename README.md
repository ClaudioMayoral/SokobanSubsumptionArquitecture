# SokobanSubsumptionArquitecture

Código fuente:
https://zetcode.com/javagames/sokoban/

1.- Clonar el repositorio de github que contiene el juego.\n
2.- Abrir el código en Visual Studio Code\n
3.- Seleccionar el archivo Sokoban.java con click derecho\n
4.- En las opciones, seleccionar “Run Java”\n
5.-Una vez que se despliegue la ventana, presionar cualquier tecla para avanzar un estado al ambiente\n
Tip : Se puede mantener presionada cualquier tecla para que el avance entre estados sea inmediato y los resultados se muestren más rápido
6.-Visualizar el resultado\n
